
## 03-08: Create a Function Componentes:

- Methode 1
```
function person() {
    return (
        <h1>I'm a Person compenent from Person/Person.js</h1>
    );
}
```

- Methode 2
```
const person = function() {

    )
}
```

- Methode 3
```
const person = () => {
    return (
        <h1>I'm a Person compenent from Person/Person.js</h1>
    );
}
```
